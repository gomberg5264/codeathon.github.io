What were all about

Internet For All wants to bring internet to everyone no matter where they live. We partnered with internet service providers to deliver satellite internet to communities that can't afford it.

This project was created for Code-a-thon 2021, by Aj Duffy, Donato Fiore, and Maxwell Gomberg. You can find the code for the site on Github and use parts of how you'd like.
